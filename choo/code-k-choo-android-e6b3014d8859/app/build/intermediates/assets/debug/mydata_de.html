Meine Daten Informationen

Bereich Meine Daten
Im Bereich meine Daten sehen Sie, welche Daten von Ihnen hinterlegt sind und für den nächsten Auftrag verwendet werden. Es werde hier immer die Daten Ihres letzten Auftrages abgebildet. Im Falle von Änderungen (z.B. neue Adresse oder Kontonummer) können Sie diese hier einfach ändern, so dass für Ihren Auftrag immer Ihre aktuellen Informationen vorliegen. Eine Änderung Ihrer persönlichen Daten kann aber ebenfalls in der Übersicht vor dem Versenden Ihres Antrages erfolgen. Daten eines versendeten Antrages können nicht mehr geändert werden.

Einreichungen
Im Bereich Einreichungen erhalten Sie eine Übersicht über alle Ihre bisher getätigten Einreichungen via Choo.

Wie verwenden wir Ihre persönlichen Daten?

Ihre persönlichen Daten sind für die Einreichung des Rückerstattungs- anspruches notwendig. Ihre Kontoinformationen benötigen wir, um Ihnen die Rückerstattung zu überweisen.

Weitere Informationen finden Sie im Menue im Bereich Informationen -> Datenschutz
